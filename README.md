## Project Period
2022.04.08 ~ 2022.04.14 (6일)
- 장문희: JWT로그인, 회원가입, 아이디 및 닉네임 중복검사
- 홍준기: 타이머 조회, 시작, 총 시간으로 저장/ 친구추가
- 김연유: 코멘트 조회, 작성, 삭제 
<br/><br/>

## Intoduction
팀원들, 학교친구들과 공부시간을 공유하며 응원할 수 있는 소셜네트워크 사이트입니다.
- 자신의 공부시간 측정하기
- 친구등록하기 (친구 공부시간, 접속중인지 확인가능)
- 등록된 친구 방명록에 코멘트 남기기
